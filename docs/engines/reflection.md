# Reflection Engine

## Overview
Extracts reflections from journal entries including insights, realizations, lessons, questions, gratitude, and growth moments.

## Features
- Rule-based extraction of reflection patterns
- Categorizes reflections by type
- Tracks reflection frequency and patterns
- Generates insights about reflection patterns

## API
- `POST /api/reflection/analyze` - Process reflections
- `GET /api/reflection/reflections` - Get reflections
