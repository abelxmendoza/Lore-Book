# Personality Engine

## Overview
Extracts personality traits from journal entries to build a personality profile.

## Features
- Detects 15+ personality traits (introverted, creative, analytical, etc.)
- Tracks trait frequency and confidence
- Identifies dominant traits
- Monitors trait evolution over time

## API
- `POST /api/personality/analyze` - Process personality
- `GET /api/personality/profile` - Get personality profile
