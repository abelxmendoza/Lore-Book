# Archetype Engine

## Overview
Extracts archetype signals from journal entries to identify dominant archetypes, transitions, and distortions.

## Features
- Detects 11 archetypes (Warrior, Rebel, Hermit, Creator, Lover, Seeker, Protector, Leader, Outlaw, Sage, Shadow_Self)
- Computes archetype profile (dominant, secondary, shadow)
- Detects transitions between archetypes
- Identifies distortions (Overdrive, Identity Split, Shadow Dominance)

## API
- `POST /api/archetype/analyze` - Process archetypes
- `GET /api/archetype/profile` - Get archetype profile
