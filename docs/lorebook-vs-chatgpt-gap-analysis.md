Date: 2026-06-15

This document is intentionally direct. ChatGPT is a mature general-purpose AI workspace. LoreBook is a more specialized and potentially more valuable product, but it must close basic interaction and trust gaps before its deeper life-memory advantages can matter.

## Highest Risk Gaps

### Trust Gap

LoreBook makes stronger claims than ChatGPT. ChatGPT can say "here is an answer." LoreBook says "this is your life." That raises the trust bar.

Required closure:

- evidence on life claims
- corrections
- memory deletion
- uncertainty labels
- source history

### Complexity Gap

LoreBook has many engines: timeline, entities, relationships, biography, analytics, memories, chapters. Users should not have to understand those systems.

Required closure:

- one primary chat input
- progressive disclosure
- simple memory review
- source-backed outputs

### Navigation Gap

ChatGPT is mostly a list of conversations. LoreBook is conversations plus memory graph plus timeline plus biography. Navigation must be better, not larger.

Required closure:

- unified search
- life projects
- thread summaries
- entity pages connected to threads

## Brutal Product Diagnosis

LoreBook should not try to be a better general chatbot. It will lose.

LoreBook should become the place where:

- the user's life record accumulates
- memory becomes inspectable
- relationships become understandable over time
- biography emerges from evidence
- AI reflection is grounded in the user's actual history

The north star is not "better chat." The north star is "I trust this system with the story of my life."

## Minimum Bar To Beat ChatGPT In This Domain

1. A user can ask "what do you remember about this?" and see sources.
2. A user can correct any wrong memory and see what changed.
3. A user can find any past conversation, event, person, or file by meaning.
4. A user can switch threads without fear of losing or corrupting context.
5. A user can create a biography chapter that cites its evidence.
6. A user can keep sensitive chats out of memory.
7. A user can inspect why the assistant personalized an answer.
8. A user can separate raw facts from interpretations.
9. A user can export their life archive.
10. A user can delete or forget information predictably.
